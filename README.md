# chat-app

## Description

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev:api

$ yarn start:dev:socket

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Environments

```bash
# System
NODE_ENV=local / development / staging / production
PORT=3000

# Database
MONGODB_URI=...
```
